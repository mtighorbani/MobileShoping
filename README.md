# MobileShoping
Mock Mobile shop

## Technologies
- core: `NextJs ("14.1.0")`, `TS`, `React`, `Redux-Toolkit`, `Axios`
- UI: `Tailwind` , `Antd`

## Scripts

- npm`i`** Install all dependencies
- npm `run build`** build the project
- npm `run dev`** run the development server
- npm `remove`** Remove dependencies
